*** INSTALLATION ***

1) npm i react-router-dom
2) npm i express
3) npm i axios
4) npm i cors
5) npm i dotenv


*** HOW TO RUN ***

git clone "URL" // Inplace of URL, type HTTP URL of this github repository

1) ./server - nodemon node.js
2) ./client - npm start

Note: Start the server first
